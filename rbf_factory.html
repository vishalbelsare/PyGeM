RBF Factory¶

Factory class for radial basis functions

class RBFFactory(fname)[source]

Bases: object

Factory class that spawns the radial basis functions.

Example
>>> from pygem import RBFFactory
>>> import numpy as np
>>> x = np.linspace(0, 1)
>>> for fname in RBFFactory.bases:
>>>     y = RBFFactory(fname)(x)
